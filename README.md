# Assessment Backend

## Tecnologies
- NestJS
- TypeORM
- PostgresSQL
- DTO (Data Transfer Object)
- Third party currency API
- Pagination
- Gateway with Web Socket IO
- Cron Job
- Repositories


## Installation

- Install Postgress
- Create Database `assessment`
- Execute `yarn`
- Execute `yarn start:dev`

## Note
- The backend serve by 8080 port.
